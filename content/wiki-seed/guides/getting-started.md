# Getting started

This page lives at path `guides/getting-started` (nested under `guides/`).

## Editing

Wiki pages open in the **Rich** Markdown editor by default (split or preview, toolbar with **Save** and **Print / PDF**). To switch to the **Classic** textarea + side preview, use **Admin → Settings → Markdown editor**. See [Markdown help](/wiki/guides/help-markdown) for details.
